# ARK Connect Demo

![ARK Connect](./banner.png)

This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

Install dependencies:

```bash
pnpm install
```

Development server:

```bash
pnpm dev
```

Production server:

```bash
pnpm build
pnpm start
```
